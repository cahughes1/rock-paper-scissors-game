# rock-paper-scissors-game
Rock Paper Scissors Game built in Python
A Python tkinter app with match history tracking and best-of-X game modes.

## Features
- Play Rock, Paper, Scissors against the computer
- Best-of-X match mode
- Game history saved to file (`rps_history.txt`)
- Input validation and error handling
- Match statistics and replay capability

## How to Run
```bash
python rps_gui.py
```

## What This Demonstrates
- File I/O and data persistence (saving/loading game history)
- Game logic and state management (best-of-X match tracking)
- Input validation and error handling (user choice validation)
- User interaction flow and decision trees
- Data formatting and display
- GUI development (tkinter)
